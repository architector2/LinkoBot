# Discord Bot для Военная-политическая-игра

Python Discord bot для вашего сервера.

## Требования
- Python 3.8+
- pip или uv

## Установка

### 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

Или с помощью uv:
```bash
uv pip install -r requirements.txt
```

### 2. Получить токен бота

1. Перейдите на [Discord Developer Portal](https://discord.com/developers/applications)
2. Нажмите "New Application"
3. Перейдите в раздел "Bot" и нажмите "Add Bot"
4. Нажмите "Copy" под TOKEN
5. Откройте файл `.env` и замените `your_bot_token_here` на ваш токен

### 3. Добавить бота на сервер

1. В Developer Portal перейдите в "OAuth2" → "URL Generator"
2. Выберите:
   - **Scopes**: `bot`
   - **Permissions**: 
     - Send Messages
     - Read Messages/View Channels
     - Manage Messages (опционально)
3. Скопируйте сгенерированный URL и откройте его в браузере
4. Выберите ваш сервер и авторизуйте бота

## Запуск бота

```bash
python bot.py
```

Вы должны увидеть:
```
✅ Bot logged in as [BotName]#0000
```

## Доступные команды

### Основные
- `!ping` - Проверить время отклика бота
- `!hello` - Поприветствовать пользователя
- `!info` - Информация о боте
- `!help` - Список всех команд

### Экономика 💰

- `!balance [@user]` - Проверить баланс (свой или другого пользователя)
- `!work` - Работать и заработать 100-500 💵 (перезарядка 1 час)
- `!daily` - Получить ежедневную награду 500 💵 (перезарядка 24 часа)
- `!pay @user сумма` - Отправить деньги другому пользователю
- `!dice сумма` - Игра в кости. Выиграешь если выпадет >50 (выигрыш = ставка × 2)
- `!leaderboard` - Топ 10 самых богатых пользователей

## Структура файлов

```
.
├── bot.py              # Основной файл бота
├── .env                # Файл с токеном (НЕ КОММИТЬТЕ!)
└── requirements.txt    # Зависимости проекта
```

## Примечания

- **Не делитесь своим токеном** ни с кем! Он как пароль для вашего бота.
- Файл `.env` находится в `.gitignore` и не должен быть закоммичен.
- Для разработки используйте тестовый сервер.

## Расширение функционала

Чтобы добавить новую команду, добавьте в `bot.py`:

```python
@bot.command(name='your_command')
async def your_command(ctx):
    """Description of command"""
    await ctx.send("Response")
```

Наслаждайтесь! 🎮
